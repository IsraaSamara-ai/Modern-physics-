Modern Physics Interactive Tool /أداة الفيزياء الحديثة التفاعلية
English Description
An interactive, single-page web application designed to simplify and visualize complex modern physics concepts for students. The tool covers the particle nature of light, including Blackbody Radiation, the Photoelectric Effect (with a step-by-step virtual lab simulation), Lenard's Experiment, Einstein's Photoelectric Equation, and the Compton Effect. Built with vanilla HTML, CSS, and JavaScript, it requires no installation and runs directly in any browser.

Features:

Interactive Blackbody Simulation: Visualize Planck's Law and spectral shifts with temperature.
Virtual Photoelectric Lab: A step-by-step guided experiment where students polish metal, charge an electroscope, and test light frequencies.
Real-time Graphs & Animations: Dynamic feedback for Lenard's and Einstein's equations.
Bilingual Interface: Arabic educational content with English mathematical symbols and equations.
Credits: Developed by Israa Youssuf Samara.

الوصف باللغة العربية
تطبيق ويب تفاعلي متكامل (صفحة واحدة) يهدف إلى تبسيط وشرح مفاهيم الفيزياء الحديثة المعقدة للطلاب. يغطي التطبيق الطبيعة الجسيمية للضوء، بما في ذلك إشعاع الجسم الأسود، والظاهرة الكهروضوئية (مع محاكاة مخبرية تفصيلية خطوة بخطوة)، وتجربة لينارد، ومعادلة أينشتين، وظاهرة كومبتون.

المميزات:

محاكاة الجسم الأسود: توضيح قانون بلانك وتأثير درجة الحرارة على اللون والطيف بصرياً.
المختبر الافتراضي للظاهرة الكهروضوئية: تجربة إرشادية تفاعلية يقوم فيها الطالب بصقل الفلز، شحن الكشاف، وتجربة أطوال موجية مختلفة وملاحظة النتائج.
رسوم بيانية ورسوم متحركة: رؤية فورية لنتائج تغيير التردد والشدة والزوايا.
واجهة ثنائية اللغة: شروحات تعليمية باللغة العربية مع الرموز والمعادلات باللغة الإنجليزية.
